🛒 Online Shopping App

A modern React-based eCommerce web application with product listing, cart system, checkout flow, and admin dashboard with order management.

✨ Features
🏠 Home page with trending products
🔍 Product search functionality
🛍️ Add to cart system 
📦 Product details page with image gallery
💳 Checkout page
🧾 Order management system (Admin)
📊 Revenue tracking (daily, weekly, monthly)
🔐 Admin panel with protected routes
☁️ Firebase integration for products & orders
📱 Fully responsive UI
